## How to
1. Fork本項目，編輯相应文件夹内的 .po 文件：
  - 將英文條目下面對應的空位（`msgstr` 那一行）填入中文翻譯即可。
2. 點擊 Pull Request 跟本項目合併（然后等待我的確認）。
  
3. （可選）也可使用 Poedit 編輯 po 文件。 [下載Poedit](https://poedit.net/)

4. 对翻译有疑问或讨论，可以点击本页面上的 `issue`，发起个 issue（相当于发帖）。
